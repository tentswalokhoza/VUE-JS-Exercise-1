# Cooking Masterclass — Catalogue (Exercise 01)

Simple Vue 3 single-page catalogue showcasing cooking courses. This project demonstrates component-driven design, a local dataset, and a wishlist interaction.

## Features
- Dynamic course listing (no duplicated markup)
- Wishlist save/un-save with header count
- Sold Out indication for unavailable courses
- Responsive card layout

## Run locally
Requirements: Node.js and npm

Install and start dev server:

```powershell
npm install
npm run dev
```

Open the URL reported by Vite (usually `http://localhost:5173`).

## Project structure
- `src/App.vue` — main app and data
- `src/components/CourseCard.vue` — course card component
- `src/components/Header.vue` — site header + wishlist count
